# resources.md - Github repos and APIs related to the project 

* metallic: Proof-of-concept GPU-accelerated drop-in replacement for numpy on macOS, powered by the Metal and Accelerate frameworks. https://github.com/jackcook/metallic

* oobabooga-macOS: information regarding how to get numerical analysis, data science, and AI core software running to take advantage of the Apple Silicon M1 and M2 processor technologies. https://github.com/unixwzrd/oobabooga-macOS

* accelerated JAX on Mac: https://developer.apple.com/metal/jax/

* Julia + Apple acceleration: has some sparse implementations: https://github.com/JuliaLinearAlgebra/AppleAccelerate.jl

* Sparce matrices on apple CPU: https://developer.apple.com/documentation/accelerate/creating-sparse-matrices

* Python wrapper for Apple Accelerate's sparse LDL^T factorization: https://github.com/bodono/macldlt

* Numerical Computing on Apple M1 : Study and Implementations: https://github.com/ShoYamanishi/AppleNumericalComputing

* rapids-singlecell: https://rapids-singlecell.readthedocs.io/en/latest/

* Techincal blog we will need to replicate: https://developer.nvidia.com/blog/gpu-accelerated-single-cell-rna-analysis-with-rapids-singlecell/
